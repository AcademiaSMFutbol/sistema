⚙️ Configuración de conexión

Para conectar con tu Google Sheet, necesitas desplegar un Web App desde Google Apps Script y pegar la URL aquí. El script tiene permisos de lectura y escritura sobre BASE DATOS SM.

Sección "Implementar como aplicación web" en Apps Script
Pasos rápidos:
1. Abre tu Sheet → Extensiones → Apps Script
2. Pega el código del archivo gas-backend.gs
3. Implementar → Nueva implementación → Aplicación web
4. Ejecutar como: Yo · Acceso: Cualquier usuario
5. Copia la URL y pégala arriba → Guardar y conectar

⚠️ Causa del error "Error de conexión":
La implementación anterior usaba POST (bloqueado por CORS). La nueva versión del GAS usa GET — debes crear una nueva implementación, no actualizar la existente.
